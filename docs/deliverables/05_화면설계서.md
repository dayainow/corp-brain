## 1. UI 설계 원칙

- **업무용 톤**: NovaPay 사내 도구에 맞는 간결한 레이아웃
- **다크 모드**: Tailwind `dark:` 클래스 지원
- **반응형**: `sm:` 브레이크포인트 기준 모바일 대응
- **접근성**: 아이콘 버튼 `aria-label`, 시맨틱 heading

## 2. SCR-001 로그인 (`/login`)

### 레이아웃

```
┌─────────────────────────────────┐
│         CorpBrain (h1)          │
│    NovaPay Internal KB          │
├─────────────────────────────────┤
│  [처음이신가요? 사용 안내]        │
│  이메일 input                    │
│  비밀번호 input                  │
│  [로그인]                        │
│  ─── 또는 ───                   │
│  [Google로 로그인] (env 설정 시) │
├─────────────────────────────────┤
│  초기 배포 계정 (원클릭 입력)     │
│  김준호 / 박수연 / 이민호 ...     │
└─────────────────────────────────┘
```

### UI 요소

| 요소 | 타입 | 설명 |
|------|------|------|
| 이메일 | `input` placeholder `name@novapay.kr` | Credentials |
| 비밀번호 | `input[type=password]` | — |
| 로그인 | `button` | NextAuth signIn |
| 시드 계정 | 버튼 목록 | 자동 입력 |

## 3. SCR-002 채팅 (`/`)

### 레이아웃

```
┌──────────┬───────────────────────────────────────────────┐
│ 사내 문서 │ CorpBrain │ NovaPay … │ [문서][도움말][Admin]… │
│ [검색…]  ├───────────────────────────────────────────────┤
│  └ 트리  │ [온보딩 배너]                                   │
│  (lg+)   │  사용자/AI 말풍선, 출처 뱃지(클릭→원문 모달)      │
│          ├───────────────────────────────────────────────┤
│          │  [질문 입력창]                      [전송]     │
└──────────┴───────────────────────────────────────────────┘
```

- **데스크톱 (`lg+`)**: 좌측 `DocumentTree` 사이드바 (폭 `w-72`)
- **모바일**: 헤더 `문서` 버튼 → 좌측 드로어 트리

### 헤더 액션 (Role별)

| 버튼 | Role | 동작 |
|------|------|------|
| 문서 | 전체 | 모바일: 트리 드로어 / 데스크톱: 사이드바 |
| 도움말 | 전체 | HelpPanel 슬라이드 |
| Admin | admin | `/admin` 이동 |
| Sync Vault | admin | POST `/api/index` |
| 업로드 | manager+ | DocumentUpload 모달 |
| Clear | 전체 | localStorage 세션 삭제 |
| 로그아웃 | 전체 | signOut |

### 채팅 메시지 스타일

| Role | 배경 | 정렬 |
|------|------|------|
| user | `bg-blue-600 text-white` | 우측 |
| assistant | `bg-white dark:bg-slate-800` + border | 좌측 |

### 출처 뱃지·원문 모달

- 패턴: `[출처: 파일명.md]`
- `ChatMessageContent`에서 **클릭 가능 뱃지** (`button`, `aria-label` 원문 보기)
- 클릭 시 `DocumentPreviewModal` — 제목·파일명·본문 (md: 마크다운, pdf/docx: 추출 텍스트)
- 닫기: `Escape`, 배경 클릭, X 버튼

### 문서 트리 검색 (사이드바)

| 요소 | 설명 |
|------|------|
| 검색 입력 | `type=search`, placeholder `문서 검색...` |
| 일치 건수 | `N건 일치` / `일치하는 문서 없음` |
| 필터 | 제목·파일명(확장자 제외) |
| 펼침 | 검색 중 매칭 폴더 자동 펼침 |
| 문서 클릭 | 해당 문서 제목으로 질문 자동 전송 |
