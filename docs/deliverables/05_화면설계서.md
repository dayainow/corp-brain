# 05. 화면 설계서 (UI 명세서)

| 항목 | 내용 |
|------|------|
| 프로젝트명 | CorpBrain |
| 문서 버전 | v1.0 |
| 작성일 | 2026-07-02 |

---

## 1. UI 설계 원칙

- **업무용 톤**: NovaPay 사내 도구에 맞는 간결한 레이아웃
- **다크 모드**: Tailwind `dark:` 클래스 지원
- **반응형**: `sm:` 브레이크포인트 기준 모바일 대응
- **접근성**: 아이콘 버튼 `aria-label`, 시맨틱 heading

---

## 2. SCR-001 로그인 (`/login`)

### 레이아웃

```
┌─────────────────────────────────┐
│         CorpBrain (h1)          │
│    NovaPay Internal KB          │
├─────────────────────────────────┤
│  [처음이신가요? 사용 안내]        │
│  이메일 input                    │
│  비밀번호 input                  │
│  [로그인]                        │
│  ─── 또는 ───                   │
│  [Google로 로그인] (env 설정 시) │
├─────────────────────────────────┤
│  초기 배포 계정 (원클릭 입력)     │
│  김준호 / 박수연 / 이민호 ...     │
└─────────────────────────────────┘
```

### UI 요소

| 요소 | 타입 | 설명 |
|------|------|------|
| 이메일 | `input` placeholder `name@novapay.kr` | Credentials |
| 비밀번호 | `input[type=password]` | — |
| 로그인 | `button` | NextAuth signIn |
| 시드 계정 | 버튼 목록 | 자동 입력 |

## 3. SCR-002 채팅 (`/`)

### 레이아웃

```
┌──────────────────────────────────────────────────┐
│ CorpBrain │ NovaPay Internal KB │ [도움말][Admin]… │
├──────────────────────────────────────────────────┤
│ [온보딩 배너 — 최초 안내]                          │
├──────────────────────────────────────────────────┤
│  (빈 상태) 빠른 질문 프롬프트 버튼                  │
│  또는                                            │
│  [사용자 말풍선 — 우측, blue-600]                   │
│  [AI 말풍선 — 좌측, white/slate-800]               │
│    - Markdown 렌더링                             │
│    - [출처: xxx.md] 뱃지                         │
│    - 👍 👎 피드백                                │
├──────────────────────────────────────────────────┤
│  [질문 입력창 — rounded-full]          [전송]      │
└──────────────────────────────────────────────────┘
```

### 헤더 액션 (Role별)

| 버튼 | Role | 동작 |
|------|------|------|
| 도움말 | 전체 | HelpPanel 슬라이드 |
| Admin | admin | `/admin` 이동 |
| Sync Vault | admin | POST `/api/index` |
| 업로드 | manager+ | DocumentUpload 모달 |
| Clear | 전체 | localStorage 세션 삭제 |
| 로그아웃 | 전체 | signOut |

### 채팅 메시지 스타일

| Role | 배경 | 정렬 |
|------|------|------|
| user | `bg-blue-600 text-white` | 우측 |
| assistant | `bg-white dark:bg-slate-800` + border | 좌측 |

### 출처 뱃지

- 패턴: `[출처: 파일명.md]`
- `ChatMessageContent`에서 클릭 가능 링크/뱃지로 렌더링
