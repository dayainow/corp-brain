## 1. 개요

본 문서는 CorpBrain 웹 애플리케이션의 화면 목록, URL, 접근 권한, 주요 기능을 정의합니다.

---

## 2. 화면 목록

| 화면 ID | 화면명 | URL | 인증 | 최소 Role | 구현 파일 |
|---------|--------|-----|------|-----------|-----------|
| SCR-001 | 로그인 | `/login` | 불필요 | — | `src/app/login/page.tsx` |
| SCR-002 | 채팅 (메인) | `/` | 필수 | general | `src/app/page.tsx` |
| SCR-003 | 사용 가이드 | `/guide` | 필수 | general | `src/app/guide/page.tsx` |
| SCR-004 | Admin 대시보드 | `/admin` | 필수 | admin | `src/app/admin/page.tsx` |

---

## 3. 화면별 요약

### SCR-001 로그인

| 항목 | 내용 |
|------|------|
| 목적 | 사내 계정 인증 후 서비스 진입 |
| 주요 기능 | Credentials 로그인, Google SSO(선택), 시드 계정 원클릭 입력 |
| 비인증 시 | 다른 화면 접근 시 본 화면으로 리다이렉트 |

### SCR-002 채팅 (메인)

| 항목 | 내용 |
|------|------|
| 목적 | 사내 문서 기반 RAG 질의·응답 |
| 주요 기능 | 스트리밍 답변, 출처 뱃지(원문 모달), **문서 트리·검색**, 도움말 패널, 문서 업로드(manager+), Sync Vault(admin), 피드백(👍/👎) |
| 세션 저장 | localStorage `chat-session` |

### SCR-003 사용 가이드

| 항목 | 내용 |
|------|------|
| 목적 | 인앱 매뉴얼·운영 안내 |
| 주요 기능 | Role별 가이드 섹션, 빠른 시작 안내 |

### SCR-004 Admin 대시보드

| 항목 | 내용 |
|------|------|
| 목적 | 운영·감사·품질 모니터링 |
| 주요 기능 | 감사 로그, 문서·청크 통계, Hit@3/MRR 메트릭 |
| 비권한 접근 | `/`로 리다이렉트 |
