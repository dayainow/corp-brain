## 2. 초기 설정

```bash
# 1. 저장소 클론
git clone https://github.com/dayainow/corp-brain.git
cd corp-brain

# 2. 의존성 설치
npm ci

# 3. 환경 변수
cp .env.example .env.local
```

### `.env.local` 필수 항목

```bash
AUTH_SECRET=<32자 이상 랜덤>   # openssl rand -base64 32
AUTH_URL=http://localhost:3000
VAULT_PATH=./vault
EMBEDDING_MODEL=Xenova/multilingual-e5-small
```

### Ollama

```bash
# 별도 터미널
ollama run llama3
```

## 3. 실행 명령

| 명령 | 용도 |
|------|------|
| `npm run dev` | 개발 서버 (`:3000`) |
| `npm run dev:fast` | fast env 프리셋 + dev |
| `npm run dev:quality` | quality env (Cross-encoder ON) + dev |
| `npm run deploy:compose` | Docker Compose 배포 |
| `npm run smoke:compose` | 파일럿 Compose 스모크 (A3~A7) |
| `npm run pilot:preflight` | D-1 사전 점검 |
| `npm run pilot:ready` | 오픈 전 전체 (Compose + E2E + RAG) |
| `npm run pilot:bday` | B-Day 계정·Slack·health |
| `npm run health:watch` | 15분 health 알림 |
| `npm run report:pilot-weekly` | D+7 주간 리포트 |
| `npm run build` | 프로덕션 빌드 |
| `npm start` | 프로덕션 서버 |
| `npm run index:vault` | Vault 전체 인덱싱 |
| `npm run index:vault -- --incremental` | 증분 인덱싱 |
| `npm run eval:search` | 검색 품질 평가 |

## 5. Docker (PgVector)

```bash
./scripts/deploy-compose.sh   # 또는 npm run deploy:compose
# 호스트 PgVector 인덱싱: config/env/compose.host.env 참고
```

### docker-compose 서비스

| 서비스 | 포트 | 설명 |
|--------|------|------|
| postgres | **5433**→5432 | pgvector (Mac 로컬 PG 충돌 회피) |
| redis | 6379 | Rate limit |
| app | 3000 | Next.js standalone |

## 9. 트러블슈팅

| 증상 | 해결 |
|------|------|
| 채팅 무응답 | Ollama 실행 확인 |
| Sync 실패 | `VAULT_PATH=./vault` |
| Health degraded | Sync Vault 실행 |
| E2E 포트 충돌 | `:3001` — `playwright.config` reuseExistingServer (로컬) |
| 임베딩 느림 | 최초 1회 HF 모델 다운로드 대기 |
