## 1. 컴포넌트 구조

```
src/
├── app/                    # 페이지 (App Router)
│   ├── page.tsx            # 채팅 메인
│   ├── login/page.tsx
│   ├── admin/page.tsx
│   └── guide/page.tsx
├── components/             # 재사용 UI
└── lib/                    # 비즈니스 로직
```

---

## 2. 페이지 컴포넌트

| 컴포넌트 | 파일 | 타입 | 설명 |
|----------|------|------|------|
| `Chat` | `app/page.tsx` | Client | 메인 채팅, useChat |
| `LoginPage` | `app/login/page.tsx` | Client | 로그인·시드 계정 |
| `AdminPage` | `app/admin/page.tsx` | Client | 대시보드 |
| `GuidePage` | `app/guide/page.tsx` | Server/Client | 매뉴얼 |

### Chat (`page.tsx`) 상태

| State | 타입 | 용도 |
|-------|------|------|
| `messages` | UIMessage[] | useChat |
| `input` | string | 입력창 |
| `isIndexing` | boolean | Sync Vault |
| `helpOpen` | boolean | 도움말 패널 |

### Chat 의존성

- `@ai-sdk/react` `useChat` + `DefaultChatTransport`
- `next-auth/react` `useSession`
- `extractMessageText` — 메시지 텍스트 추출
